Friends Router Documentaion:



*GET*


/favorites -
query: {
  index: i
}

Fetches from database a user's favorites only.
Starts from i, and goes until i + 11 (MAX: 12).
Returns an array of objects in order newest to oldest.


*POST*


/favorites/add -
body: {
  uuid: i,
  gifId: j,
  ...
}

Post to database a single gif that a user favorited.
Returns an object of the gif which was just favorited.
